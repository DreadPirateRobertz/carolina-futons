10. Illustration Style Guide

Blue Ridge Mountain Reference

Soft rolling ridgelines — 5-7 overlapping layers
fading into distance with atmospheric haze
design.jpeg is the quality bar

Quality Requirements

  • 1. 5+ gradient stops for sky transitions
  • 2. 15+ distinct elements per scene (trees, birds, flowers, smoke, stars)
  • 3. All colors from sharedTokens.js (zero hardcoded hex)
  • 4. Atmospheric depth: FG/MG/BG layers with varying opacity
  • 5. design.jpeg visual comparison PASS required
  • 6. Figma Draw only — no coded SVG template strings

Approved Brush Styles

Pencil Tool

Mountain ridgeline sketching. Smooth, flowing paths that follow the rolling terrain of the Blue Ridge. Variable width for natural feel.

Brush Tool

Watercolor texture fills. Soft, feathered edges for sky washes and mountain face shading. Layer multiple strokes for depth.

Scatter Brush

Foliage distribution, wildflower meadows, birds in flight, stars. Random scatter with controlled density creates organic feel.

Figma Effects for Illustration

Texture Effect

Edge distressing for hand-made paper feel. Apply to illustration borders for authentic watercolor edge treatment.

Noise Effect

Paper grain texture overlay. Subtle 3-5% noise on solid fills creates tactile warmth. Essential for non-digital feel.

Progressive Blur

Atmospheric haze for distant mountain layers. Blur increases with distance: 0px foreground, 2px midground, 4-8px background.

Dynamic Strokes

Hand-drawn wiggle on vector paths. Converts precise lines to organic, imperfect strokes that feel human-made.

Do / Don't

Do

Use soft, rolling ridgelines with 5-7 overlapping mountain layers. Each layer slightly lighter and more blurred than the one in front. Colors from the brand palette only.

Don't

Use sharp, geometric mountain shapes or flat solid-color triangles. No programmatic feTurbulence/feDisplacementMap SVG filters — they produce abstract, not illustrative results.

Do

Design in Figma Draw with Pencil, Brush, and Scatter tools. Export as optimized SVG. Inject brand token colors before integration.

Don't

Write SVG as JavaScript template strings with inline filter effects. This approach is deprecated — it can't match hand-drawn quality.

Do

Include 15+ distinct natural elements: varied trees, birds, wildflowers, cabin smoke, stars, clouds. Layer foreground, midground, and background at different opacities.

Don't

Use fewer than 5 elements per scene. Avoid symmetrical, computer-perfect arrangements. Nature is organic and imperfect — embrace that.

Do

Use warm, espresso-tinted shadows (rgba(58,37,24,...)). Warm gradients for sky (blue-to-gold). Sand and coral warmth throughout.

Don't

Use cold gray shadows, pure black outlines, or neutral/clinical colors. This site has CHARACTER — generic minimalist furniture-site template is failure.
