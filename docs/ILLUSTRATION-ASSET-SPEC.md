# Carolina Futons — Illustration Asset Specification

## Design Direction: Blue Ridge Mountain Illustrative

**Style Reference**: See `~/Desktop/design.jpeg` for the approved mockup aesthetic.

The visual identity blends warm, hand-drawn illustration with a modern furniture
e-commerce experience. Think: mountain landscape postcards meets Anthropologie
catalog — warm, inviting, intentional.

---

## Color Palette (all illustrations must use these)

| Color | Hex | Usage |
|-------|-----|-------|
| Sand Base | `#E8D5B7` | Sky fills, paper texture backgrounds |
| Sand Light | `#F2E8D5` | Highlight areas, clouds |
| Sand Dark | `#D4BC96` | Terrain shadows, mountain mid-tones |
| Espresso | `#3A2518` | Outlines, linework, darkest shadows |
| Espresso Light | `#5C4033` | Secondary linework, softer outlines |
| Mountain Blue | `#5B8FA8` | Sky accents, distant ridgelines, water |
| Mountain Blue Light | `#A8CCD8` | Upper sky, atmospheric haze |
| Sunset Coral | `#E8845C` | Sun, warm glow, accent highlights |
| Sunset Coral Light | `#F2A882` | Sunset gradients, warm reflections |
| Mauve | `#C9A0A0` | Tertiary accents, flowers, soft details |
| Sky Top | `#B8D4E3` | Top of sky gradient |
| Sky Bottom | `#F0C87A` | Horizon glow (sunset scenes) |

**Linework**: Hand-drawn feel, slightly irregular strokes. NOT perfectly geometric.
Use `#3A2518` (espresso) for primary outlines, `#5C4033` for lighter secondary lines.

## Asset 1: Mountain Ridgeline Header (PRIORITY: HIGH)

**Used on**: Every page (sticky header background, announcement bar area)
**Format**: SVG (scalable, transparent) + PNG fallback @2x
**Dimensions**: Full-width (1440px wide, ~120px tall), must tile seamlessly
**Description**: A continuous Blue Ridge Mountain ridgeline silhouette that sits
behind the navigation bar. Multiple overlapping ridge layers creating depth.

**Layers (back to front)**:
1. Farthest ridge — `#A8CCD8` (mountain blue light), very soft/faded
2. Middle ridge — `#5B8FA8` (mountain blue), moderate detail
3. Nearest ridge — `#D4BC96` (sand dark) with tree line, most detail
4. Subtle tree silhouettes along nearest ridge (pines, varied heights)

**Key requirements**:
- Must tile horizontally (left edge connects seamlessly to right edge)
- Transparent background (sits over sand-colored header)
- No sun/sunset elements — just ridgelines
- Evokes the view from Hendersonville looking toward the Blue Ridge Parkway

## Asset 2: Hero Illustration — Mountain Cabin Scene (PRIORITY: HIGH)

**Used on**: Homepage hero section
**Format**: SVG preferred, PNG @2x acceptable
**Dimensions**: 1440 x 600px (desktop), responsive crop for mobile
**Description**: A cozy mountain cabin/showroom nestled in the Blue Ridge foothills,
viewed from a slight distance. Warm light glows from windows. This is the
"Handcrafted Comfort, Mountain Inspired." hero image.

**Scene elements**:
- A charming wooden cabin/shop building (not too literal — slightly idealized)
  with a welcoming front porch
- Warm golden light spilling from windows (sunset coral glow)
- Blue Ridge Mountain ridgeline behind the cabin
- Sunset/golden hour sky with gradient from `#B8D4E3` top to `#F0C87A` horizon
- A simple sun partially behind the mountains (using `#E8845C`)
- Scattered trees (mix of deciduous and evergreen, it's Western NC)
- Foreground: natural elements — a path, wildflowers, rocks
- Optional: subtle smoke from a chimney (warmth, handcrafted feel)

**Mood**: Warm, welcoming, family-owned. Like coming home.

**Text overlay area**: Left half of the image should have a clear area (sky/simple
background) where the hero title "Handcrafted Comfort, Mountain Inspired." will
be overlaid in `#3A2518` Playfair Display.

## Asset 3: Illustrated Map — Hendersonville Area (PRIORITY: MEDIUM)

**Used on**: Contact page, About page, footer
**Format**: SVG + PNG @2x
**Dimensions**: ~800 x 600px
**Description**: A hand-drawn illustrated map showing the store location in
Hendersonville, NC, with surrounding landmarks for context.

**Map elements**:
- Store pin/building at 824 Locust St — highlighted, warm glow
- Nearby landmarks labeled in hand-lettered style:
  - Blue Ridge Parkway
  - Pisgah National Forest
  - Downtown Hendersonville (Main Street)
  - I-26 corridor
  - Asheville (directional indicator, ~25 miles)
  - Lake Lure / Chimney Rock (directional)
- Mountain terrain illustrated in the background
- Roads shown as simple hand-drawn lines
- Trees and terrain features scattered appropriately
- A compass rose in one corner

**Style**: Like a vintage park map or an illustrated wine country guide.
Not a Google Maps screenshot — a piece of art.

## Asset 4: Category Card Illustrations (PRIORITY: HIGH)

**Used on**: Homepage category showcase, shop navigation
**Format**: PNG @2x (with transparent or sand-colored background)
**Dimensions**: 400 x 300px each (6 illustrations)
**Description**: Individual product category illustrations, each showing
a representative piece of furniture in the illustrative style.

### 4a. Futon Frames
- An elegant wooden futon frame in the open (sofa) position
- Show the quality craftsmanship — visible joinery, warm wood grain
- A couple of throw pillows for scale/warmth
- Background: subtle mountain vista through a window

### 4b. Futon Mattresses
- A plush futon mattress shown at an angle, emphasizing thickness/quality
- Fabric texture visible (not flat/boring)
- Cross-section callout showing foam layers (optional but nice)
- Clean, inviting bedroom/living room setting

### 4c. Murphy Cabinet Beds
- A Murphy cabinet bed in CLOSED position (looks like a cabinet)
- Inset/ghost showing it OPEN as a bed (or a before/after split)
- Show the transformation magic — it's a cabinet AND a bed
- Modern room setting

### 4d. Platform Beds
- A solid wood platform bed frame with clean lines
- Show the slat system briefly (the 2.8" spacing for memory foam support)
- Warm bedroom setting with mountain view window

### 4e. Casegoods & Accessories
- A curated arrangement: nightstand, dresser, small chest
- All matching wood finish/style
- Styled with small decorative items (lamp, plant, book)

### 4f. Sale / Clearance
- A playful "SALE" themed illustration
- Maybe a price tag with mountain/sunset design
- Furniture pieces with sale starburst tags
- Energetic but still matching the warm, hand-drawn aesthetic

## Delivery Specifications

**File naming convention**:
```
cf-asset-[number]-[name]-[variant].svg
cf-asset-[number]-[name]-[variant]@2x.png
```

Examples:
```
cf-asset-01-ridgeline-header.svg
cf-asset-02-hero-cabin@2x.png
cf-asset-04a-category-futon-frames@2x.png
```

**Color mode**: sRGB for all web assets
**File size targets**: SVGs < 50KB each, PNGs < 500KB each (optimize for web)
**Transparency**: SVGs should have transparent backgrounds unless noted
